# AGENTS

This project packages a tk-driven implementation workflow for pi.

## Core instructions

- Read and follow `.tf/AGENTS.md` for reusable lessons and workflow conventions.
- Use the `tk` command line tool for ticket operations (`tk ready`, `tk show <id>`, `tk blocked`, `tk close <id>`).
- Use `/tk-bootstrap` to install/update subagent templates from this package.
- Use `/tk-implement <ticket-id>` for ticket execution with re-anchoring and persistent knowledge behavior.
